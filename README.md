# 🧮 Calculadora em Python

Uma calculadora simples desenvolvida em Python, capaz de realizar operações básicas.

## 🚀 Funcionalidades:

Soma +
Subtração -
Multiplicação *
Divisão /

## 💻 Como executar o projeto

1. Instale o Python na sua máquina

2. Clone este repositório:

```
git clone https://github.com/tamaradasilva/calculadora.git
```

3. Acesse a pasta do projeto:

```
cd calculadora
```

4. Execute o arquivo:

```
python calculadora.py
```

## 📌 Requisitos

* Python 3 ou superior

## ✨ Sobre

Projeto simples para prática de lógica de programação com Python.
